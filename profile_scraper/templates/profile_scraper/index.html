Twitter Classifier

We classify tweets from a given Twitter user in two types business and non-business. We take information available through a LeadSift database CSV dump to generate a list of user candidates. From this list of candidates we use text pre-processing (punctuation, digits and stopwords) and a weighting algorithm to qualify tweets.
